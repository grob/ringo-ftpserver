Module events

The default registry/dispatcher module used for listening on ftp server events and dispatching them to registered listeners.

Events.prototype.callListeners (eventName, args)

Calls all registered callbacks for a given event name

Parameters

String eventName The name of the event
Array args An array containing the arguments to pass to the callbacks

Returns

org.apache.ftpserver.ftplet.FtpletResult If one of the callbacks returns false, this method returns FtpletResult.SKIP, otherwise it returns FtpletResult.DEFAULT

org.apache.ftpserver.ftplet.FtpRequest ftpRequest The FTP request

Returns
